# Operators: Comparison
## Truthy / Falsy 

<pre class="code javascript" >
// length != 0 == true
if ( items.length ) 
  console.log( 'has stuff' );

// length == 0 == false
else
  console.log( 'is empty' );
</pre>

* 'Truthy' is when a value _acts_ like it's `true`
* 'Falsy' is when a value _acts_ like it's `false`
