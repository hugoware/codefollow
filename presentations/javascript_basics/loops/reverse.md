# Loops
## Reverse

<pre class="code javascript" >
var fruit = [ 'apple', 'orange', 'pear' ];
for ( var i = fruit.length; i-- &gt; 0;  )
  console.log( i, fruit[i] );

// 2 'pear'
// 1 'orange'
// 0 'apple'
</pre>

* you do not need to follow the pattern exactly, there are multiple ways to achieve the same result
